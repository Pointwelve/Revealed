## GraphQL Update Schema
1. Install Apollo CLI `npm install -g apollo`
2. run `apollo schema:download --endpoint="YOUR_END_POINT" --header="YOUR_AUTH_TOKEN"`
3. Update the schema in `Revealed/GraphQL/schema.json`.
